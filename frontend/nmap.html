📋 NMAP COMMANDS

Click any command to copy

🔍 BASIC SCANS

  • nmap 127.0.0.1
    Simple scan on localhost
  • nmap -sV 192.168.1.1
    Version detection on target
  • nmap -O 192.168.1.1
    OS detection (requires sudo)
  • nmap -A 192.168.1.1
    Aggressive scan (OS, version, traceroute)

🌐 NETWORK SCANS

  • nmap -sn 192.168.1.0/24
    Discover all devices on network (ping sweep)
  • nmap -sL 192.168.1.0/24
    List scan - DNS resolution only
  • nmap 192.168.1.1-100
    Scan IP range 1 to 100

🔌 PORT SCANS

  • nmap -p 22,80,443 192.168.1.1
    Scan specific ports
  • nmap -p- 192.168.1.1
    Scan ALL ports (1-65535) - slow
  • nmap -p 1-1000 192.168.1.1
    Scan first 1000 ports
  • nmap --top-ports 100 192.168.1.1
    Scan top 100 most common ports

⚡ FAST & STEALTH

  • nmap -F 192.168.1.1
    Fast scan (100 ports only)
  • nmap -T4 -F 192.168.1.1
    Fast scan with aggressive timing
  • sudo nmap -sS 192.168.1.1
    Stealth SYN scan (requires sudo)

🎯 OTHER USEFUL

  • nmap -sV --script vuln 192.168.1.1
    Vulnerability scan (slow)
  • nmap -sU -p 53 192.168.1.1
    UDP scan on DNS port
  • nmap -oN output.txt 192.168.1.1
    Save output to file
